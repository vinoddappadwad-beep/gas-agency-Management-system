# Gas Agency Management System

This is a web-based Gas Agency Management System project.

## Features
- Customer Management
- Gas Booking System
- Cylinder Delivery Tracking
- Admin Panel

## Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript

## How to Run
1. Install XAMPP
2. Move project to htdocs folder
3. Import database.sql in phpMyAdmin
4. Run project on localhost
